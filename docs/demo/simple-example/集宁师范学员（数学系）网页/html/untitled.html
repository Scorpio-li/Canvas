数学系简介： 1977年9月成立乌盟师范学校大专班，并于当年招收第一届大专生，1983年在原大专班理科组的基础上成立乌盟教育学院数学系,1985年更名为乌盟师专数学系，1995年更名为集宁师范高等学校数学系，2009学校晋升本科，更名为集宁师范学院数学系。1992年开始招收三年制专科生,2009开始招收四年制本科生，是学校建系最早、发展最快的教学系之一。 数学系现有数学与应用数学、信息与计算科学两个本科专业；数学教育专科专业，现有592名在校生，教职工25名，专兼职教师31名，其中“双师”型教师7名，高级职称10人，博士1人，硕士16名，在读博士1人。现有函数论、学科教学论、信息与计算科学、代数与几何、大学数学等5个教研室。 数学系已为内蒙古自治区内外和乌兰察布市培养了3100多名合格人才，其中大多数已成为自治区和乌兰察布市数学教学的主干力量。以我系学生为主的大学生数学建模代表队每年参加全国大学生数学建模比赛，2004、2005年连续两年获全国二等奖及多项赛区奖；2008、2009年连续两年获全国一等奖及多项赛区奖（乙组）；2011年到2014年均获得赛区一等奖（甲组）两次，赛区二等奖（甲组）两次，多个成功参赛奖（甲组）。2013年、2014年我系数学与应用数学专业毕业生有19人考取内蒙古大学、内蒙古师范大学、内蒙古工业大学等学校硕士研究生。 我系还与区内外高校建立了良好的学术交流往来关系，聘请复旦大学数学科学学院教授、博士生导师袁小平，上海海事大学文理学院教授、博士生导师特木尔朝鲁为我系客座教授。 近年来我系教师在各类刊物发表论文100余篇,获一项自治区教学成果二等奖，主（参）编自治区十一五规划教材9部，数学教育专业被评为自治区品牌专业，《数学分析》、《解析几何》课程被评为自治区级精品课程，《大学数学与数学建模教学团队》被评为自治区级优秀教学团队。目前我系承担的主要省级科研项目共6项，经费达28万元。
